# バイブコーディングキャンプ｜デモリポジトリ

Claude Code セットアップ講義で使う共通リポジトリです。

## このリポジトリについて

Next.js + Supabase + Google認証 + Stripe 構成のサンプルプロジェクトです。  
実際のアプリ開発前のセットアップ練習用として使います。

## 使い方

```bash
git clone https://github.com/kotashimizu/vibe-coding-camp-demo.git
cd vibe-coding-camp-demo
```

AntigravityでOpen Folderを使って開いてください。

## 講義の流れ

1. このリポジトリをクローン
2. Antigravityのターミナルで `claude` を起動
3. `claude-code-setup` プラグインを導入
4. プロジェクト分析・提案を受け取る
5. CLAUDE.md を確認する
